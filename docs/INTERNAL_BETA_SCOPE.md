# ShortFlow Studio Premiere 내부 베타 범위

기준일: 2026-07-11

## 목표

내부 베타의 목적은 **Premiere 안에서 반복 편집 시간을 실제로 줄이는 로컬 도구**를 만드는 것입니다. 판매, 마케팅, SaaS 운영을 위한 기능은 현재 합격 기준에 포함하지 않습니다.

목표 기간은 한 달입니다. Mock Host·정적 계약·자동 테스트·빌드는 로컬 후보를 검증하고, 실제 Host smoke test와 타임라인 삽입 검증은 별도 게이트로 수행합니다. Mock 통과를 실제 Premiere 승인으로 대체하지 않습니다.

## 포함 범위

| 영역 | 내부 베타 범위 |
|---|---|
| Host bridge | 공개 Premiere UXP API 경계, 버전·capability guard, 비파괴 mutation, Mock Host 테스트 |
| 자막/STT/SRT | 안정적인 `cueId`/`wordId`, 단어 시간, SRT 입출력, 분할·병합·숨김·줄바꿈, undo/redo, 프로젝트별 autosave |
| TTS | 기본 생성, 로컬 저장, 미리듣기 가능한 파일, 타임라인 삽입 경계 |
| 음악/SFX | 폴더 열기, 기존 파일 동기화, 폴더/카테고리, 드래그 순서 이동, 미리듣기, 타임라인 삽입 |
| 썸네일 | 1280×720 Canvas, 1~4분할, 이미지 이동·확대·크롭, 텍스트·배지·그림자·외부광선, 밝기·대비·채도, PNG/JPG 로직. Premiere 26.3 Host에서 Canvas export가 제한되면 SVG fallback 저장을 임시 대체 경로로 사용하며 PNG/JPG Host 승인은 별도 보류 |
| Safe Zone | 플랫폼·역할·revision이 표시되는 보수적 오버레이와 기본 정렬 |
| 자동 편집 | STT 기반 무음 컷 추천 마커와 기본 펀치인. 원본 보존·중복 실행 차단 |
| 안정성 | 설정, 자동 저장, 복구 저널, 사용자 실행 로컬 진단 로그, 민감정보 제거 |
| 에셋 권리 | 출처, 라이선스, 상업 사용 가능 여부, 만료일, 출처 표기, 내보내기 전 경고, 권리 리포트 |
| 검증/패키징 | typecheck, lint, test, build, dist/CCX 구조 검증, 내부 베타 체크리스트 |

## AI 이미지·영상 허용 범위

- AI 레퍼런스 보드
- 외부에서 만든 이미지·영상 파일 가져오기
- 프롬프트 메모·태그·출처 기록
- 프로젝트 폴더 기반 정리
- 에셋 권리 정보 기록

이미지·영상 생성 모델 호출, 전체 생성 파이프라인, 고급 provider orchestration은 내부 베타 범위가 아닙니다.
썸네일 AI 보정(GPT Image 2 기본·강렬 보정, 화질 개선, 배경 제거, 대화형 수정)은 2026-07-13 사용자 지시로 내부 베타 UI에서 활성화했습니다(후순위 #4·#5·#6 당김, `docs/01-plan/features/deferred-ai-features.plan.md`). A/B 자동 판단, 이미지·영상 생성 파이프라인, BGM 비트 매칭·덕킹은 후속 단계입니다.

## 에셋 권리 필수 데이터

각 음악·이미지·외부 AI 산출물·TTS 결과 에셋은 다음 정보를 기록할 수 있어야 합니다. 이는 출처와 사용권 추적 범위이며, 내부 베타에 AI 이미지·영상 생성 파이프라인을 포함한다는 뜻은 아닙니다.

- 출처
- 라이선스
- 상업 사용 가능 여부: 가능 / 불가 / 미확인
- 만료일(선택 가능하되, 존재하면 유효성 검사)
- 출처 표기 문구
- 마지막 수정 시각과 에셋 식별자

내보내기 전에는 누락, 상업 사용 불가, 미확인, 만료된 권리를 경고하고 JSON/Markdown 권리 리포트를 생성해야 합니다. 권리 판단을 자동으로 보증하지 않으며 사용자가 최종 확인합니다.

## 후순위 범위

- 결제·라이선스·플랜 제한
- 상용 SaaS 계정·서버·결제
- 자동 텔레메트리 서버
- AI 이미지·영상 생성 전체 파이프라인
- 썸네일 AI 대화 수정·A/B 자동 판단
- 고급 BGM 비트 매칭·자동 덕킹
- 다국어 패키지
- 스마트 리프레임·피사체 추적
- 플랫폼별 업로드 패키지 자동 생성

## 로컬 내부 베타 후보 완료 조건

- 포함 범위의 핵심 경로가 패널 진입점 또는 명시적 Mock Host 어댑터에 연결되어 있습니다.
- 사용자 파일·원본 시퀀스를 손상시키는 실패 경로가 자동 테스트로 차단됩니다.
- 권리 정보 누락·불가·미확인·만료 경고와 리포트가 동작합니다.
- `npm run typecheck`, `npm run lint`, `npm test`, `npm run build`가 모두 성공합니다.
- 내부 베타 패키징 체크리스트와 재현 가능한 산출물 검증이 완료됩니다.
- 중간 커밋 없이 검증된 체크포인트 커밋 하나를 만듭니다.

## Premiere 설치 후 실제 Host 승인 조건

Mock Host 통과는 실제 Premiere 동작을 보증하지 않습니다. 설치 후 [실제 Host Smoke Runbook](HOST_BETA_RUNBOOK.md)의 12개 필수 항목과 다음 환경 경계를 별도로 확인해야 합니다.

- 패널 로드·재로드·닫기와 listener/timer 수명주기
- 프로젝트·시퀀스 전환과 프로젝트별 autosave
- 파일 권한·persistent token·한글/공백 경로
- 오디오·자막·가이드·펀치인 삽입과 원본 보존
- 실제 CCX 설치와 호스트 버전 호환성
